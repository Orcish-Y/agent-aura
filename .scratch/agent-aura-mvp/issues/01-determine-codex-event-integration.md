# Determine the reliable Codex event integration

Type: research
Status: open
Blocked by:

## Question

What supported Codex CLI integration can pass stable Thread identity, working directory, title, lifecycle, approval/input waiting, success, failure, interruption, and liveness signals from arbitrary locally launched Codex TUI sessions to Agent Aura without forcing users to launch Codex through Agent Aura? Compare command hooks, plugins, app-server/remote-control subscriptions, and local session records; identify version constraints, event-to-domain mappings, security boundaries, and a reversible automated install/repair/uninstall flow.
